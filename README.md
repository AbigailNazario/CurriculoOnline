**Currículo Online - HTML e CSS**

Este projeto foi desenvolvido como atividade da disciplina Coding: Mobile.
O objetivo foi criar um currículo online utilizando HTML e CSS, aplicando conceitos básicos de estruturação de páginas e estilização.

**Objetivo do Projeto**

Desenvolver uma página web que apresente um currículo profissional de forma clara e organizada, utilizando apenas HTML para estrutura e CSS para estilização.

**Tecnologias**

HTML5

CSS3

**Estrutura do Projeto**

O projeto é composto por arquivos simples que formam a página do currículo:

index.html – Estrutura principal da página

style.css – Arquivo responsável pela estilização do currículo

**Funcionalidades**

Apresentação de informações pessoais

Seção de formação acadêmica

Experiências ou projetos

Organização visual utilizando CSS

Layout simples e responsivo

**Aprendizados**

Com este projeto foi possível praticar:

Estruturação de páginas com HTML

Estilização com CSS

Organização de conteúdo em uma página web

Criação de um projeto simples para portfólio no GitHub
